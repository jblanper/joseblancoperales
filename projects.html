These are projects I have been doing during my spare time. At the moment, all the projects reflect my interest in producing and manipulating images in the browser with the canvas api. Some are more interactive and others more artistic. Please enjoy them!

This project links to three small apps that allow the user to upload and manipulate images in different ways. Some are just for fun (like Pixelator) and others can be more practical.

This project links to two small interactive experimental apps where the user can produce interesting designs. The app called Floroid generates a random machine each time the page is loaded which resembles a spirograph, but more complicated. The other app, Sine globe (name to be revised), allows the user to change the form of a moving "blob" of ellipses by manipulating some parameters. The sliders are a little unpredictable.

This project contains several animated sketches of moving forms that resemble marine animals in their motion. The code is based on Sine globe, but the animations are not interactive. It is recommended to use Chrome to visualize these animations. They run very slow in mobile browsers, so they are better enjoyed in a desktop.
